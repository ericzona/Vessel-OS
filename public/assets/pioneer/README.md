# Pioneer Asset Layers (0-15)

Each subdirectory corresponds to a LayerSlot:
- 0: Background
- 1: Shadow
- 2: Body (Skin Tone)
- 3: Eyes
- 4: Shirt
- 5: Pants
- 6: Shoes
- 7: Jacket/Outerwear
- 8: Hat/Headwear
- 9: Glasses/Eyewear
- 10: Necklace/Jewelry
- 11: Badge/Pin
- 12: Tool/Weapon (Right Hand)
- 13: Tool/Weapon (Left Hand)
- 14: Aura/Glow Effect
- 15: Particle Effects

Place PNG assets here. HUD will fallback to colored blocks if missing.
